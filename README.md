# 📈 Cryptocurrency Price Analysis

## 📌 Project Overview

The Cryptocurrency Price Analysis project focuses on analyzing historical cryptocurrency market data to identify trends, price movements, and market behavior. Using Python and data analysis techniques, the project provides insights through data visualization and statistical analysis.

## 🎯 Objectives

- Analyze historical cryptocurrency price data.
- Identify market trends and patterns.
- Visualize price fluctuations over time.
- Compare the performance of different cryptocurrencies.
- Generate insights for data-driven decision-making.

---

## 🛠️ Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

---

## 📂 Dataset

The dataset contains historical cryptocurrency market data, including:

- Date
- Open Price
- Close Price
- High Price
- Low Price
- Trading Volume
- Market Capitalization

## 🚀 How to Run

### Clone the Repository

```bash
git clone https://github.com/your-username/cryptocurrency-price-analysis.git
```

### Navigate to the Project Directory

```bash
cd cryptocurrency-price-analysis
```

### Install Dependencies

```bash
pip install pandas numpy matplotlib seaborn
```

### Run the Notebook

```bash
jupyter notebook
```

Open the notebook and execute all cells.
